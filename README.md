# 🌸 FlowerClicker - 꽃심어서 뒷돈벌기 💰

> 미술수업 제출용 (근데 진짜 재밌음)

**클릭만 하면 돈이 나온다!** 진짜로! 마우스만 톡톡 두드리면 돈이 쏟아집니다. 꽃도 심고, 새 알도 잡고... 게임인지 현실인지 모를 정도로 중독적입니다. ⚠️ 주의: 시간 가는 줄 모릅니다.

## 🎮 이게 뭔데요?

그냥... 클릭하면 돈 나오는 게임입니다. 간단하죠? 근데 꽃 심으면 자동으로 돈이 나오고, 새가 알 떨어뜨리면 그거 잡으면 엄청난 돈이 나옵니다.

## ✨ 뭐가 있나요?

### 💰 클릭 = 돈
- 화면 아무데나 클릭하세요. 돈 나옵니다.
- 업그레이드하면 클릭 한 번에 돈이 더 많이 나옵니다.
- 마우스 버튼이 아까워질 수도 있습니다 (아님)

### 🌺 꽃 = 자동 돈 기계
- 꽃 심으면 알아서 돈을 만들어줍니다. 진짜 착하죠?
- 꽃들이 화면 아래에서 왔다갔다 합니다. 귀여워요.
- 업그레이드하면 꽃 하나당 돈이 더 많이 나옵니다.
- 꽃 100개 심으면... 상상해보세요 🌸🌸🌸

### 🐦 새 = 보너스 스테이지
- 가끔 새가 날아옵니다. 알을 떨어뜨립니다.
- 그 알을 클릭하면 **엄청난 돈**이 나옵니다!
- 놓치면 후회합니다. 진짜요.
- 새가 왜 알을 떨어뜨리는지는... 비밀입니다 🤫

## 🚀 시작하는 법 (엄청 쉬움)

1. `index.html` 열기 (더블클릭하면 됩니다)
2. 마우스로 화면 톡톡 두드리기
3. 돈 모이면 업그레이드하거나 꽃 심기
4. 끝! 이제 중독되셨습니다 🎉

## 🎯 꿀팁 (진짜 꿀팁)

- 💡 처음엔 클릭 업그레이드부터! 돈 모으기 쉬워집니다
- 🌸 꽃 많이 심으면 나중에 편합니다. 진짜로
- 🥚 새 알 놓치지 마세요! 한 번 놓치면 다음 기회까지 기다려야 합니다
- 📈 꽃 수익 업그레이드는 장기 투자입니다. 믿고 투자하세요
- ⏰ 시간 정해놓고 하세요. 안 그러면 밤새게 됩니다
